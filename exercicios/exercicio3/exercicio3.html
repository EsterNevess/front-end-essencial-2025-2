Adicionando Itens ao Carrinho com `.push()`

O array carrinho começa vazio e adicionamos 3 itens a ele.

Código JavaScript:

const carrinho = [];

// Usando .push() para adicionar múltiplos itens de uma só vez
carrinho.push(
    { nome: "Mouse Gamer", preco: 150 },
    { nome: "Teclado Mecânico", preco: 300 },
    { nome: "Monitor 24'", preco: 850 }
);

console.log("Carrinho Final:", carrinho);
